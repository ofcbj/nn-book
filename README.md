# Neural Network Visualizer

## 概要 (Overview)

このプロジェクトは、**nn.pdf**の「組織・面接のメタファー」を用いて、ニューラルネットワークの動作原理を視覚的に理解できるインタラクティブWebアプリケーションです。

PyTorchなどの外部ライブラリを使わず、**純粋なJavaScript**で実装されており、ブラウザだけで動作します。

## 特徴 (Features)

### 🧠 ネットワーク構造
- **入力層**: 5ニューロン (応募者の属性)
- **隠れ層1**: 8ニューロン (第一次面接官)
- **隠れ層2**: 6ニューロン (第二次面接官) 
- **隠れ層3**: 4ニューロン (最終面接官)
- **出力層**: 1ニューロン (意思決定者)

### ✨ 主な機能
- **リアルタイム可視化**: ニューロン間の重みを色と太さで表現
- **インタラクティブ学習**: スライダーで入力値を調整し、即座に結果を確認
- **順伝播・逆伝播**: アニメーションで学習過程を可視化
- **統計情報**: 損失関数、エポック数、学習率をリアルタイム表示

### 🎨 デザイン
- モダンなダークテーマ
- Glassmorphismエフェクト
- スムーズなアニメーション
- レスポンシブデザイン

## 使い方 (Usage)

### 1. ローカルサーバーの起動

```bash
# Python 3がインストールされている場合
cd c:\src\nn-book
python -m http.server 8000
```

### 2. ブラウザでアクセス

```
http://localhost:8000
```

### 3. 操作方法

#### 入力の調整
- 5つのスライダーで入力値 (0.0 ~ 1.0) を調整
- 目標出力値も調整可能

#### 学習の実行
- **1ステップ**: 1回だけ学習を実行
- **学習開始**: 連続的に学習を実行 (損失が0.001以下になると自動停止)
- **リセット**: ネットワークを初期状態に戻す

#### 表示設定
- **重み表示**: ニューロン間の接続線を表示/非表示
- **活性化値表示**: 各ニューロンの出力値を表示/非表示
- **バイアス表示**: バイアス項の表示/非表示

## 実装の詳細 (Implementation Details)

### ニューラルネットワーク
- **活性化関数**: Sigmoid
- **損失関数**: Mean Squared Error (MSE)
- **最適化**: Gradient Descent
- **学習率**: 0.1 (デフォルト)

### 行列演算
すべての行列演算 (multiply, transpose, add, subtract, hadamard) を自前実装。

### 可視化
- Canvas APIを使用した動的レンダリング
- 粒子システムによるアニメーション
- 重みの値に応じた色・太さの変化

## メタファーの対応関係

| ニューラルネット概念 | 組織・面接のメタファー |
|---------------------|---------------------|
| 入力ベクトル | 応募者の情報・属性 |
| ニューロン | 面接官・評価者 |
| 重み | 評価基準・価値観 |
| 内積 | 信頼度の数値化 |
| 活性化関数 | 判断の表現 |
| 層 | 意思決定の階層 |
| 順伝播 | 評価プロセス |
| 誤差逆伝播 | 責任の配分・反省 |
| 学習 | 信頼構造の更新 |
